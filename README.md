# My Portfolio

Welcome to the source code for my personal portfolio website! 

## About This Project
This is a responsive, modern portfolio built using HTML, CSS, and Vanilla JavaScript. It highlights my skills as a developer, my featured projects, and provides easy ways to get in touch. The portfolio uses a dynamic JSON structure to effortlessly manage content and includes elegant scroll animations for a premium user experience.

## Live Demo
🌐 **View the live portfolio here:** [https://Yildirim28.github.io/MyPortfolio/](https://Yildirim28.github.io/MyPortfolio/)

## Key Features
- **Dynamic Content**: Projects and skills are dynamically loaded from a central `data.json` file.
- **Interactive Filtering**: Easily filter projects by category (Web, Fullstack, Data, etc.) with a single click.
- **Smooth Animations**: Intersection Observers trigger smooth fade-in and slide-up animations as elements scroll into view.
- **Fully Responsive**: Carefully designed to look great on desktop, tablet, and mobile devices.
- **Dark Mode Support**: Built-in theme toggling between light and dark modes, saving user preferences to `localStorage`.

## Technologies Used
- **HTML5** & **CSS3** (Custom Properties, Flexbox, Grid)
- **Vanilla JavaScript** (Fetch API, DOM Manipulation, IntersectionObserver)
- **FontAwesome** (Scalable vector icons)
